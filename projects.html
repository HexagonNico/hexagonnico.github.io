MARK'S KEEP

March 2022 - Now

Mark's Keep is the project I am currently working on, a game heavily inspired by A Link to the Past made using the Godot engine.

For this game I am trying to apply everything I learned about level design to create intricate levels that require the player to explore to find their way.

I have been keeping track of the progress I make on the project by recording devlog videos for my youtube channel. Keeping this video-diary is helping me to stay motivated to keep working on the project.

VECMATLIB

May 2022

I originally created this project for my programming course in university. Several months later I came back to finish the project and publish a finished version.

VecMatLib is a Java library that provides simple structures to represent vectors and matrices and implement all their basic operations.

The project is now hosted on Github and is available for everyone to use.

INVAZORS

March 2022

Invazors was a project created for a game jam. The theme of the jam was "roles reversed" and was implemented in the game by reversing the roles of a tower defense game, in which your objective is to command an army of UFOs to destroy the enemy's tower.

This game was the first 3D game I created in Unity. Creating it taught me how 3D games are handled in the engine and the differences in the implementation between 2D games and 3D games.

MYTHIC PIECES

January 2022

I created Mythic Pieces to learn to use the Godot engine. After creating a few test project to learn the basics, I decided to join a game jam to create an actual game with it.

The theme of the jam was "fitting in". My implementation of the theme was a puzzle game in which the player must move around pieces of the map to create a path to the end of the level.

REWORKED METALS

November 2021

Reworked Metals is a mod I created for Minecraft, which changes the metalworking system of the game.

Modding games helped me improve my games design abilities by making improvements to already existing systems using the hooks that are provided to me by the game.

After completing the project, I also kept maintaining for a while by using the Github issue tracker.

DOORS OF FATE

July 2021

Doors of Fate was a game created for a game jam about time. "Time is never on our side. But what if time can be manipulated? What if we can control time?"

After reading the question asked by the theme of the jam I decided to create a puzzle-horror game where the player must change its system's time to solve puzzles. The strong suit of the game was its dark and mysterious atmosphere and its meta-narration.

Created using the Unity engine.

UNLUCKIEST RUINS

June 2021

This project was created for a game jam with the theme "falling apart". My implementation of the theme was a platformer game every platform collapses beneath the player's feet.

With this game I tried to apply what I had learned about the level design of platform games: start by letting the player experiment with the game mechanics in a safe environment and build up on them as the level progresses.

Created using the Unity engine.

SEAL OF THE FOX

February - November 2021

Seal of the Fox was the first long term project I created using the Unity engine.

It is a metroidvania game inspired by Hollow Knight and Ori, in which I tried to create intricate levels in which the player could get lost.

A few prototypes for the game were uploaded on itch.io with the names "Fox game prototype" and "Fox game demo", although the project is currently on hold for an indefinite time.

BUZZY DRONES

March 2021

Buzzy Drones was a Minecraft mod I created after playing Slime Rancher. I tried to port a feature of the second game into the first one.

Modding games helped me improve my games design abilities by adding new features to a game using the hooks that are provided to me by the game.

PIXEL ISLAND

March 2021

This was a project I originally created to learn to use the Unity engine. After the overwhelmingly positive response it had on itch.io I decided to turn it into a completed game.

Pixel Island is a game with no challenge or objective about building a village on an island. It is a simple and relaxing game that lets players express their creativity through a few simple actions.

DUNGEON KEYS

October 2020 - January 2021

Dungeon Keys was my first ever long-term project, a dungeon crawler game I created with my own engine.

Developing my own engine for this game helped me understand how engines work behind the scenes and taught me how to work on the same project for a long time.

Created using Java and LWJGL.

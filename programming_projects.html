VECMATLIB

May 2022

I originally created this project for my programming course in university. Several months later I came back to finish the project and publish a finished version.

VecMatLib is a Java library that provides simple structures to represent vectors and matrices and implement all their basic operations.

The project is now hosted on Github and is available for everyone to use.

HEXAGON ENGINE

January 2022 - April 2022

An unfinished general-purpose game engine written in Java with LWJGL.

This project was mainly a side project that I started because my most recent games all used already existing engines and I missed having to create my own as I did for my first projects. I decided at some point to switch from Java to Unity/Godot because creating an engine from scratch for every game was too time consuming.

Although the project was left unfinished, I may start working on it again in the future.

MAZE GENERATOR

August 2020

A Java implementation of the Aldous-Broder algorithm for randomly generating a maze.

I recorded this tutorial video to explain how I implemented this algorithm in one of my previous projects.

PERLIN NOISE WORLD GENERATOR

February 2020

A Java implementation of a 2D world generator that uses Perlin Noise.

The program is able to render a 2D map of a randomly generated fantasy world.

I created this project for a tutorial video for my youtube channel in which I explained some basic information about random generation and periodic functions.

CAVE GENERATOR WITH CELLULAR AUTOMATON

January 2020

This program randomly generates a top-down view of a cave system using cellular automaton.

The algorithm is implemented in Java and explained in a video I recorded for my youtube channel.

PREDATOR AND PREY - LIFE SIMULATION

December 2019

This program can simulate the life of two species, predators and preys, by applying a simple set of rules, showing a graphical representation of the environment as well as a graph showing the number of predators and preys at any given time.

Written in Java with the Swing library.

CONWAY'S GAME OF LIFE

November 2019

A Java implementation of the cellular automaton known as "Conway's Game of Life".

This program can simulate the life of simple organisms by applying a limited set of rules.

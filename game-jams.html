The following projects are games I created for Game Jams, game design marathons were developers have to created a game that fits a given theme in a short span of time. Since when I started participating in game jams, my abilities as a designer have greatly improved, before then I was just a programmer. The improvement can clearly be seen by looking at the projects below in a chronological order.

Mythic Pieces

Mythic Pieces was created for the Curdle Jam 21, a non-ranked jam with the theme "fitting in". I had the idea of a puzzle game where the player must fit together different pieces of the map in order to reach the end of the level.

This game was also my first Godot Engine release. Learning how to use the engine was my primary scope for this project.

Release date: 8th January 2022

  • Store page: Itch.io
  • Source code: Github

Doors of Fate

Doors of fate was created for a jam about time, the theme was "Time is never on our side, but what if it was? What if we could control time?", with the optional theme "escaping fate". The way the theme was presented made me think of a psychological-horror game where the player must solve puzzles by literally manipulating their system's clock.

The game was well received and was ranked 32nd out of 104 entries. The entry received the most points for technicality and creativity.

Release date: 12th July 2021

  • Store page: Itch.io
  • Submission page: Itch.io

Unluckiest Ruins

Unluckiest Ruins was created for the Low Stakes High-Rolling 2 jam, where the theme was "falling apart". My entry is a platform game where all platforms collapse beneath the player's feet, resulting in a fast gameplay where the player must think and react quickly if they want to reach the end of the level.

The game was ranked 2nd out of 4 entries. It received the most point for entertainment and implementation of the theme.

Release date: 26th June 2021

  • Store page: Itch.io
  • Submission page: Itch.io

Tower of Time

Tower of Time was created for GameDev.tv Community Jam, with the theme "time". The game I submitted is a tower defense game where time is both the currency spent to create defences and the player's health. I created this game before I knew how to use any game engine, so I built one from scratch for it using LWJGL.

Not only this game was my first entry to a game jam, it was also the first game I publicly released. It was ranked 182nd out of 227 entries.

Release date: 29th May 2020

  • Submission page: Itch.io

Dungeon Keys
← Home

Dungeon Keys was my first mid-term project. The game is written in Java, using a custom game engine that I built from scratch using LWJGL.

I was responsible for all of the programming and all of the art for the game. When working on this project I learned a lot about engine programming, how to write solid systems for my game and how to optimize my code. Drawing the art also taught me the basis of pixel art.

Despite the significant effort required to create a game engine from scratch I am satisfied with the end result. Developing this game helped me improve my commitment to reaching long-term goals.

The following are most of the projects I created to learn programming and game development. I kept a playlist on Youtube with all my games as a journal to record all the progress I made during these years, progress which can be clearly seen if observing the projects from the oldest to the most recent.

Most of these projects are Java games, some are games written in C++, others are life simulations made using Cellular Automaton and some are tutorial videos where I try to explain some topics related about certain algorithms.

Almost every single one of these projects is open source and their code can be seen by browsing through my older git repositories.

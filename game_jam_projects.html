INVAZORS

March 2022

Invazors was a project created for a game jam. The theme of the jam was "roles reversed" and was implemented in the game by reversing the roles of a tower defense game, in which your objective is to command an army of UFOs to destroy the enemy's tower.

This game was the first 3D game I created in Unity. Creating it taught me how 3D games are handled in the engine and the differences in the implementation between 2D games and 3D games.

MYTHIC PIECES

January 2022

I created Mythic Pieces to learn to use the Godot engine. After creating a few test project to learn the basics, I decided to join a game jam to create an actual game with it.

The theme of the jam was "fitting in". My implementation of the theme was a puzzle game in which the player must move around pieces of the map to create a path to the end of the level.

DOORS OF FATE

July 2021

Doors of Fate was a game created for a game jam about time. "Time is never on our side. But what if time can be manipulated? What if we can control time?"

After reading the question asked by the theme of the jam I decided to create a puzzle-horror game where the player must change its system's time to solve puzzles. The strong suit of the game was its dark and mysterious atmosphere and its meta-narration.

Created using the Unity engine.

UNLUCKIEST RUINS

June 2021

This project was created for a game jam with the theme "falling apart". My implementation of the theme was a platformer game every platform collapses beneath the player's feet.

With this game I tried to apply what I had learned about the level design of platform games: start by letting the player experiment with the game mechanics in a safe environment and build up on them as the level progresses.

Created using the Unity engine.

TOWER OF TIME

May 2020

Tower of Time was my first entry to a game jam, created using a game engine I wrote from scratch during the jam period.

The theme of the jam was "Time". My implementation of the theme was a tower defense game where time functions both as your health and as the game's currency.

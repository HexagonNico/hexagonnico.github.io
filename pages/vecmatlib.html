VecMatLib

Project information

  • Category: Software tools
  • Project date: May 2022 / December 2022
  • Project URL: github.com/HexagonNico/VecMatLib

A Scala library that provides data structures to represent vectors and matrices

I originally created this project for my Java programming course in university, it was called JLAL at the time. Several months later I came back to improved the project and publish the definitive version. Several more months later I decided to re-write the project in Scala to make use of its operator overloading feature.

VecMatLib provides data structures that represent vectors and matrices and allows the user to compute several operations involving them, such as matrix multiplications and matrix-vector products.

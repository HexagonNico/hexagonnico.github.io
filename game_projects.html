MARK'S KEEP

March 2022 - July 2022

Mark's Keep is the project I am currently working on, a game heavily inspired by A Link to the Past made using the Godot engine. The first demo was released on July 25.

For this game I am trying to apply everything I learned about level design to create intricate levels that require the player to explore to find their way.

I have been keeping track of the progress I make on the project by recording devlog videos for my youtube channel. Keeping this video-diary is helping me to stay motivated to keep working on the project.

SEAL OF THE FOX

February - November 2021

Seal of the Fox was the first long term project I created using the Unity engine.

It is a metroidvania game inspired by Hollow Knight and Ori, in which I tried to create intricate levels in which the player could get lost.

A few prototypes for the game were uploaded on itch.io with the names "Fox game prototype" and "Fox game demo", although the project is currently on hold for an indefinite time.

PIXEL ISLAND

March 2021

This was a project I originally created to learn to use the Unity engine. After the overwhelmingly positive response it had on itch.io I decided to turn it into a completed game.

Pixel Island is a game with no challenge or objective about building a village on an island. It is a simple and relaxing game that lets players express their creativity through a few simple actions.

DUNGEON KEYS

October 2020 - January 2021

Dungeon Keys was my first ever long-term project, a dungeon crawler game I created with my own engine.

Developing my own engine for this game helped me understand how engines work behind the scenes and taught me how to work on the same project for a long time.

Created using Java and LWJGL.
